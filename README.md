### 🔁 Automated Outreach + Reply Tracking System
`n8n` · `HubSpot CRM` · `Resend API` · `OpenRouter LLM` · `Webhooks`

- LLM generates personalized background summary per contact
- LLM writes a custom HTML outreach email per person
